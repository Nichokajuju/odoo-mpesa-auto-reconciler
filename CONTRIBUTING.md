# Contributing

Pull requests are welcome. For major changes, please open an issue first.

## Setup
1. Clone the repo into your Odoo custom addons folder
2. Requires Odoo 18 with Accounting module installed
3. Run with `python3 odoo-bin -c your.conf -d yourdb -u mpesa_auto_reconcile`
